# BSH Portfolio — Tangible Intelligence

변서희(Byun Seohee) 포트폴리오 · 스마트 피지컬 제품 / 산업디자인.
Static site (HTML/CSS/JS), deployed via GitHub Pages.

**Live:** https://anollity-inc.github.io/BSH_Portfolio/

## Structure
```
index.html        — single-page portfolio
css/style.css     — editorial monochrome design system
js/main.js        — scroll reveal + nav
assets/img/…      — project imagery
.nojekyll         — serve files as-is (no Jekyll processing)
```

## Local preview
Open `index.html` in a browser, or serve the folder:
```
python -m http.server 8000   # http://localhost:8000
```
